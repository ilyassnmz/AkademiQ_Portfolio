# 🚀 Portfolio Project – ASP.NET Core MVC

Bu proje, **AkademiQ.Ai** sürecinde verilen case çalışmaları kapsamında geliştirilmiş,  
**Full Stack bir Portfolyo Web Uygulamasıdır**.

Uygulama; kişisel bilgilerin, projelerin, deneyimlerin, eğitimlerin, sertifikaların ve mesajların **Admin Panel** üzerinden yönetilebildiği dinamik bir yapıya sahiptir.

---

## 🛠️ Kullanılan Teknolojiler

- **ASP.NET Core MVC**
- **Entity Framework Core (DB First)**
- **Microsoft SQL Server**
- **HTML, CSS, JavaScript**
- **ViewComponent**
- **CRUD (Create, Read, Update, Delete)**

---

## 📌 Özellikler

- Admin Panel üzerinden içerik yönetimi
- Proje, deneyim, eğitim, sertifika ve mesaj yönetimi
- Dinamik ve genişletilebilir mimari
- Temiz ve modüler kod yapısı
- Responsive (mobil uyumlu) arayüz
- Sidebar aktif menü sistemi
